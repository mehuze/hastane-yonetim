
# Hasta Takip Sistemi

C# ve SQL Server ile geliştirilmiş Windows Forms tabanlı hasta takip uygulaması.

## Özellikler
- Kullanıcı giriş ve kayıt sistemi
- Hasta ekleme, güncelleme, silme (CRUD)
- Bölüm ve durum yönetimi
- DataGridView ile hasta listeleme

## Kullanılan Teknolojiler
- C# (.NET Framework 4.7.2)
- Windows Forms
- SQL Server
- Stored Procedure
- ADO.NET

## Kurulum
1. SQL Server'da `Db_HastaneYonetim` veritabanını oluşturun
2. Projeyi Visual Studio ile açın
3. `frmSqlBaglanti.cs` dosyasında sunucu adınızı güncelleyin
4. Projeyi çalıştırın
